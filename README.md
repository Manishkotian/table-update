# Table-Update

## How to setup

### Clone the project

Run the command
 ### `git clone https://github.com/Manishkotian/table-update.git`
 
 Go to project directory
 ### `cd table-update`
 
 ### Package install

Run the command
 ### `npm install`
 ### `npm run installDependencies`


## How To Run

Run the command
 ### `npm start`

Note: 
To upload file select `sample.txt` in root folder
